## About this project

![GitHub API](https://img.shields.io/badge/GitHub%20REST%20API-181717?style=for-the-badge&logo=github&logoColor=white) ![REST API](https://img.shields.io/badge/REST%20API-0052CC?style=for-the-badge&logo=json&logoColor=white)

Utility software in Python that connects directly to GitHub servers to manage actions in real time through secure HTTP requests.


I built this project to practice working with REST APIs and to better understand how internet traffic and HTTP requests work.

## How to run:

First, you need to generate a personal access token.

Go to:
Profile → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)

Then generate a new token and use it as your password when authenticating requests.

## Features

- Follow GitHub users
- Unfollow GitHub users
- List followers
- List following users
- Authenticate using Personal Access Tokens
- Send requests directly to the GitHub REST API
